Como utilizar branches

Branch Master ou main é o braço principal em um repositório.
Branch secundária é o braço secundário em um repositório.

O comando branch é literalmente uma ramificação. Enquanto mantemos o código de produção na principal, podemos criar outras ramificações para trabalhar livremente nesse código e somente depois fazer o merge, ou seja, juntarmos os códigos.

Realização de Hospedagem de sites

Para realizar a hospedagem, é necessário um provedor de hospedagem de site para fornecer espaço em um servidor, além da armazenação de todos os arquivos e banco de dados.

Sempre que alguém digitar o domínio no navegador, o provedor transfere os arquivos necessários para atender a solicitação.
