TAG Table

A tag table representa a criação de tabelas para o design de uma homepage, fazendo alinhamentos que não seriam possíveis de fazer apenas com comandos simples.

O comando para se inserir uma tabela é table ,para colocar a linha introduzimos a tag tr e para uma coluna dt.

Sintaxe:

Nome Idade Curso
Carolina 17 anos Técnico em Informática

TAG Del

A tag del define um texto que foi deletado de um documento.
Esse elemento é renderizado pelos navegadores com uma linha entre o texto.

Sintaxe: Esse texto foi excluído.

TAG Ins

A tag ins define um texto que foi inserido em um documento.
Os navegadores sublinham o texto inserido.

Sintaxe: Minha matéria preferida é geografia história.

TAG Sub

A tag sub define um texto que foi subscrito.
O texto subscrito aparece por meio caractere abaixo da linha normal, renderizado em uma fonte menor.

Sintaxe: Esse texto foi subscrito.

TAG Sup

A tag sup define um texto sobrescrito.
O texto sobrescrito aparece por meio caractere acima da linha normal, renderizado em uma fonte menor.

Sintaxe: Esse texto foi subscrito.

TAG Blockquote

A tag blockquote espeficica uma seção citada de outra fonte.
Geralmente, os navegadores recuam os elementos.

Sintaxe:

Blockquotes são seções citadas de outras fontes.

TAG Q

A tag q define pequenas citações.
Geralmente, os navegadores inserem aspas entre aspas.

Sintaxe: Uma frase importante: A resposta certa, não importa nada: o essencial é que as perguntas estejam certas. Mario Quintana.

TAG Abbr

A tag abbr define uma abreviatura ou acrônimo, como HMTL, CSS, Sr, ou DR.

Sintaxe: ONU se estabeleceu em 24 de outubro de 1945.

TAG Address

A tag addres define as informações de contato com o autor de um documento.
As informações de contato podem ser em email, URL, endereço físico, número de telefone, entre outros.
O texto é renderizado em itálico e os navegadores sempre adicionarão uma quebra de linha antes e depois.

Sintaxe:

carolsachser@gmail.comEmail para contato.

TAG Cite

A tag cite define o título de um trabalho criativo, sendo um livro, poema, filme ou uma música. O elemento é renderizado em itálico.

Sintaxe: O Rei Leão por Disney, produzido em 1994.

TAG Bdo

A tag bdo é usada para substituir a direção do texto atual.

Sintaxe: O texto irá da direita para a esquerda.

TAG Code

A tag code é usada para definir um pedaço de código de computador.
O conteúdo interno é exibido na fonte monoespaçada padrão do navegador.

Sintaxe: O HTML define um botão clicável.

TAG Pre

A tag pre é usada para definir o texto pré-formatado.
O texto é exibido em uma fonte de largura fixa, preservando os espaços e as quebras de linhas.

Sintaxe: 

O texto é exibido em uma fonte de largura fixa preservando os espaços e as quebras de linhas.

TAG Kbd

A tag kdp é usada para definir a entrada do teclado.
O conteúdo interno é exibido na fonte monoespaçada padrão do navegador.

Sintaxe: Pressione Ctrl + C para copiar o texto.
