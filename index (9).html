Softwares de controle

Tipos de softwares de controle de versões

Centralizado
Possuem um repositório central em um servidor com controle de versões. É DEPENDENTE do servidor
Distribuído
Tem um repositório local com controle de versões. As atualizações são feitas no repositório central e são INDEPENDENTES do servidor

Ambientes para a criação de repositórios

Ambientes como GitHub ou o Git são os principais aplicativos onde criamos repositórios que podem ser linkados em qualquer aplicativo de linguagem HTML
