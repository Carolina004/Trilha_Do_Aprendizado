Multimídias.

É uma forma de comunicação que combina diferentes formas de conteúdo, como texto, áudio, imagens ou animações de vídeo em uma única apresentação.

TAG Iframe.

É usada para incorporar outro documento ao documento HTML.
Sintaxe:

Tag Vídeo.

Essa tag é usada para incorporar o conteúdo de um vídeo em um documento, como um clipe de filme ou outros fluxos de vídeo.
Essa tag contém uma ou mais tags com diferentes fontes de vídeo,o navegador escolherá a primeira fonte que suporta.

TAG Áudio.

É usada para incorporar o conteúdo de som em um documento, como músicas ou outros fluxos de áudio.
Essa tag contém uma ou mais tags com diferentes fontes de áudio,o navegador escolherá a primeira fonte que suporta.

TAG Picture.

A tag oferece aos desenvolvedores web mais flexibilidade na especificação de recursos da imagem.
O elemento contém duas tags: uma ou mais tags source e uma tag img.
O navegador procurará o primeiro elemento source onde a consulta de mídia corresponde à largura da janela de visualização atual,e em seguida, exibirá a imagem adequada. O elemento img é necessário como o último filho do elemento img, como uma opção de fallback se nenhuma das tags de origem corresponder.
